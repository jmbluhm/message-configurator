System Actions Syntax

Actions are comma-separated and wrapped in brackets. Commas inside brackets are preserved.

  • Single action: [Action name]
  • Multiple actions: [Action 1],[Action 2]

Example:
[Calls API endpoint],[Creates user profile],[Updates database, with transaction]
